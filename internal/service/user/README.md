# user service 目录

## 目录说明
用户相关业务逻辑处理模块。

## 功能描述
- 用户注册、登录、注销
- 密码管理（修改、重置）
- 用户信息管理
- 权限验证和角色管理
- 用户配额管理

## 主要文件
- **user_service.go** - 用户服务接口定义
- **user_service_impl.go** - 用户服务实现
- **auth_service.go** - 认证服务
- **role_service.go** - 角色权限服务

## 核心功能
- 用户生命周期管理
- JWT认证和授权
- RBAC权限控制
- 密码安全处理
- 用户状态管理
