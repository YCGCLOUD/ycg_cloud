# utils 目录

## 目录说明
通用工具函数模块，提供项目中常用的工具函数。

## 功能描述
- 字符串处理工具
- 时间处理工具
- 加密解密工具
- 文件操作工具
- 网络工具（HTTP、IP、URL处理）
- 验证工具（仅邮箱验证）
- HTTP响应工具

## 主要文件
- **string.go** - 字符串处理工具
- **time.go** - 时间处理工具
- **crypto.go** - 加密解密工具
- **file.go** - 文件操作工具
- **http.go** - HTTP请求工具
- **ip.go** - IP地址处理工具
- **url.go** - URL处理工具
- **validator.go** - 数据验证工具
- **response.go** - HTTP响应工具

## 工具函数分类
- **字符串工具**: 随机字符串、格式化、转换、特殊字符处理
- **时间工具**: 时间格式化、时区处理、时间计算、时间校验
- **加密工具**: MD5、SHA256、BCrypt、AES、JWT相关工具
- **文件工具**: 文件类型检测、大小格式化、MD5计算、路径处理
- **网络工具**: HTTP请求、IP地址解析、URL处理和校验
- **验证工具**: 邮箱验证、参数校验（不包含手机号验证）
- **响应工具**: 统一的API响应格式、错误码定义、分页响应

## 使用原则
- 纯函数设计，无副作用
- 完善的单元测试覆盖
- 详细的文档注释
- 性能优化
